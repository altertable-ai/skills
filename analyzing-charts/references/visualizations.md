# Visualizations Guide

How to read and interpret different visualization types.

## Line Charts

### Reading the Chart

- **X-axis**: Usually time
- **Y-axis**: Metric value
- **Line**: Trend over time
- **Multiple lines**: Comparison

### Key Observations

#### Trend Direction
- Upward slope = growth
- Downward slope = decline
- Flat = stability

#### Slope Steepness
- Steep = rapid change
- Gradual = slow change
- Changing slope = acceleration/deceleration

#### Volatility
- Smooth line = consistent
- Jagged line = volatile
- Sudden changes = events/anomalies

### Questions to Ask

1. What's the overall direction?
2. Are there seasonal patterns?
3. Where are the peaks and troughs?
4. Are there sudden changes?
5. How do lines compare?

## Bar Charts

### Reading the Chart

- **X-axis**: Categories
- **Y-axis**: Values
- **Bar height**: Magnitude
- **Bar color**: Grouping (if applicable)

### Key Observations

#### Distribution Shape
- Uniform = evenly distributed
- Skewed = concentration on one end
- Bimodal = two peaks

#### Outliers
- Unusually tall bars
- Unusually short bars
- Missing categories

### Questions to Ask

1. Which category is largest/smallest?
2. How much larger is the leader?
3. Is the distribution expected?
4. Are there outliers?

## Area Charts

### Reading the Chart

- Similar to line chart
- Filled area shows volume
- Stacked areas show composition

### Key Observations

#### Stacked Areas
- Total height = sum
- Band width = contribution
- Changing proportions = shifting composition

#### Single Area
- Area under curve = total volume
- Good for emphasizing magnitude

### Questions to Ask

1. How is total volume changing?
2. How is composition shifting?
3. Which segment is growing/shrinking?

## Pie Charts

### Reading the Chart

- **Slice size**: Proportion
- **Labels**: Categories
- **Total**: 100% of something

### Key Observations

#### Dominance
- One large slice = concentrated
- Even slices = distributed
- Many small slices = fragmented

#### Changes Over Time
- Compare pie charts across periods
- Look for shifting proportions

### Limitations

- Hard to compare similar sizes
- Too many slices = unreadable
- Best for 3-6 categories

### Questions to Ask

1. Is any segment dominant?
2. Are proportions as expected?
3. Are there surprisingly large/small segments?

## Tables

### Reading the Table

- **Rows**: Individual records or groups
- **Columns**: Different metrics
- **Sorting**: Current ordering

### Key Observations

#### Patterns
- Sorted columns reveal rankings
- Cross-column patterns
- Groupings and clusters

#### Data Quality
- NULL values
- Unexpected values
- Duplicates

### Analysis Techniques

1. Sort by different columns
2. Look for outliers
3. Check for missing data
4. Identify correlations

## Metric Cards

### Reading the Card

- **Large number**: Current value
- **Comparison**: Change indicator
- **Context**: Previous period, target

### Key Observations

#### Direction
- ▲ Up = increase (good/bad depends on metric)
- ▼ Down = decrease
- → Flat = no change

#### Magnitude
- Small % = minor change
- Large % = significant change

### Questions to Ask

1. Is the change significant?
2. Is the direction good or bad?
3. How does it compare to target?

## BarList (Ranked List)

### Reading the Chart

- **Items**: Ranked from top
- **Bar length**: Relative size
- **Labels**: Values and percentages

### Key Observations

#### Concentration
- Long first bar = one dominant item
- Gradual decline = distributed
- Sharp drop-off = head-heavy

#### Long Tail
- Many small items at bottom
- Consider grouping as "Other"

### Questions to Ask

1. How concentrated is the top?
2. What's in the long tail?
3. Are rankings expected?

## Funnel Charts

### Reading the Chart

- **Width**: Number at each step
- **Narrowing**: Drop-off
- **Percentages**: Conversion rates

### Key Observations

#### Biggest Drop-offs
- Where does funnel narrow most?
- Is this expected?
- What causes the drop?

#### Overall Conversion
- Top to bottom ratio
- Industry benchmarks
- Historical comparison

### Questions to Ask

1. Where is the biggest drop-off?
2. Is overall conversion acceptable?
3. How does it compare to benchmarks?
